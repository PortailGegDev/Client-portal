Afin d'avoir accès à vos données quotidiennes de consommation dans votre espace client, vous devez autoriser le distributeur de gaz à transmettre à GEG vos données de consommation pour le contrat sélectionné.
ces données sont stockées 3 ans. A tout moment, vous avez la possibilité de retirer votre autorisation dans la rubrique "Mes préférences" de votre espace client.

Si vous activez cette option gratuite, vos données quotidiennes seront disponibles sur votre espace client dans quelques jours
